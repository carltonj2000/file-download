# File downloader with React and Express

To run do the following in two terminals.

```
cd frontend
npm run start
cd backend
npm run start
```

## Code Generation Steps

### Frontend

```
npx create-react-app frontend
cd frontend
npm start
```

### Backend

```
npm install express-generator -g
express backend
cd backend
npm install
npm start
```
